CSV-Datei "names.csv" beinhaltet eine Auflistung von Namen, Bundesstaaten der USA, Geschlecht, die Jahre 1910 - 2010 und die Anzahl,
wie oft der Name in dem entsprechenden Jahr in dem jeweiligen Bundesstaat vergeben wurde.
Die Spalten lauten: ID / Name / Year / Gender / State / Count

Der Code "Name Counter" filtert den häufigsten Namen und die Anzahl aus der kompletten Liste.
